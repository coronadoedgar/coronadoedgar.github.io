  1. Descargar y descomprimir odoo9 {% highlight python linenos %} {% raw %} wget http://nightly.odoo.com/9.0/nightly/src/odoo_9.0.latest.tar.gz tar -xzvf odoo_9.0.latest.tar.gz {% endraw %} {% endhighlight %}
  2. Instalar librerías python {% highlight python linenos %} {% raw %} sudo aptitude update sudo aptitude install libxml2-dev libxslt1-dev libldap2-dev libsasl2-dev libssl-dev libjpeg-dev libfreetype6-dev zlib1g-dev libpng12-dev sudo aptitude install postgresql-server-dev-X.Y {% endraw %} {% endhighlight %}
  3. Instalar requerimientos de odoo {% highlight python linenos %} {% raw %} sudo pip install -r requirements.txt sudo pip install --upgrade Pillow {% endraw %} {% endhighlight %}
  4. Instalar NodeJS y less
    **Installing Node.js via package manager {% highlight python linenos %} {% raw %} curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash - sudo apt-get install -y nodejs sudo npm install -g less sudo npm install -g less-plugin-clean-css {% endraw %} {% endhighlight %}
  5. Crear archivo de configuración

    Dentro del src crear un archivo odoo-server.conf con el siguiente contenido:

    {% highlight python linenos %} {% raw %} [options] db_host=False db_port=False db_user=odoo db_password=odoo addons_path=/PATH/addons {% endraw %} {% endhighlight %}
  6. Ejecución {% highlight python linenos %} {% raw %} python odoo.py -c odoo-server.conf {% endraw %} {% endhighlight %}
