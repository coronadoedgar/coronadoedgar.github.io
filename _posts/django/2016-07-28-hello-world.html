Prerequisito

  • Virtualenv
  • Python 3.5.x
  • Django 1.9.x

Tips

  1. Localización del directorio site-packages python:
    python -c "import sys, pprint; pprint.pprint(sys.path)"
  2. Localización del directorio Django:
    import django
    django.__path__
    django.VERSION
  3. Obetner Django
    pip install Django==1.9.7

Crear proyecto

  1. django startproject misitio
  2. cd misitio && python manage.py migrate
  3. python manage.py createsuperuser
  4. python manage.py runserver
