¿Porqué usar Polymer?

  • Recicla tu código.
  • Modulariza tu app.
  • Separa tu aplicación en componentes.
  • Reusa otros componentes del trabajo de otras personas.
  • Comunidad que desarrolla muchos custom-elements .

Instalar Polymer CLI

  1. Instalar NodeJS https://nodejs.org/en/download/package-manager/ {% highlight ruby linenos %} curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash - apt-get install -y nodejs {% endhighlight %}
  2. Instalar Bower {% highlight ruby linenos %} npm install -g bower {% endhighlight %}
  3. Instalar Polymer CLI {% highlight ruby linenos %} npm install -g polymer-cli {% endhighlight %}

Más info:

https://www.polymer-project.org/1.0/docs/tools/polymer-cli

Crear proyecto elemento.

{% highlight ruby linenos %} 1) crear carpeta: polymer/edg-element 2) ingresar a la carpeta: polymer init 3) seleccionamos la opción: element 4) ingresar nombre, descripción. 5) iniciar el servicio: polymer server, en mi caso estoy trabajando con una máquina virtual (https://coronadoedgar.wordpress.com/2016/06/07/configurar-firewall-en-ubuntu/), por ello modifico el parámeto host: polymer serve --hostname '0.0.0.0' 6) en local ingreso desde el puerto 5685: http://localhost:5685/components/edg-element/ {% endhighlight%}
