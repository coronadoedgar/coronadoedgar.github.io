  1. Descargar Opencv: http://opencv.org/downloads.html

  2. Copiar en el directorio: C:\OpenCV-version

    Luego ingresar a: C:\OpenCV-version\opencv\build\python\2.7\x86

    Copiar cv2.pyd a C:\Python27\Lib\site-packages


  3. Agregar a las variables del sistema, C:\OpenCV-version\opencv\build\x64\vc12\bin

  4. Para desarrollar aplicaciones con OpenCV utilizaremos Visual Studio Community, verificamos Python Environments.

  5. Crear aplicación Python.

  6. A continuación los requisitos que serán necesarios para empezar a programar.
